# SQL Data Cleaning & Analytics Project

## Overview
This project demonstrates SQL data cleaning and business analytics using SQL Server.

## Dataset
- customers.csv
- orders.csv

## Data Cleaning Tasks
- Identified NULL values
- Removed duplicate records
- Standardized city and state names
- Replaced missing values
- Validated data quality

## Analytics Performed
- Total Sales Analysis
- Top Customers Analysis
- Monthly Sales Trend
- Order Status Analysis

- SQL Server
- SSMS
- GitHub

- GROUP BY

- CASE Statements
- Data Cleaning
